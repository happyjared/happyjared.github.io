Hexo官网

什么是Hexo

  Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other languages) and Hexo generates static files with a beautiful theme in seconds. ( Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。)

准备Node和Git环境

  1. 到Node官网下载对应系统的安装包，如果是Windows系统安装也很简单，下载最新版本的msi安装包，一路点击Next就行，会自动添加Path环境，安装完了之后按Windows窗口键+R，输入cmd打开命令提示符界面，用node -v测试即可（Tip：使用如下的命令可以更改为国内淘宝的NPM镜像：npm install -g cnpm –registry=https://registry.npm.taobao.org， 这样以后就可以使用 cnpm 命令来安装模块了，cnpm install ）

  2. 安装Git和配置好Git环境，这里就不在赘述了，安装完了可以用git –version来测试

安装hexo，推荐官方文档

  1. 输入cnpm install hexo -g，开始安装Hexo
  2. 输入hexo -v，检查hexo是否安装成功
  3. 创建一个文件夹，在当前文件夹打开命令行界面，输入hexo init，开始初始化当前文件夹，初始化完了可以看到最后输出的一句是：“Start blogging with Hexo！”
  4. 输出cnpm install，安装依赖的组件
  5. 输入hexo g（等价于hexo generate），开始生成Hexo
  6. 输入hexo s（等价于hexo server），开启本地服务，通过访问本地默认的4000端口网址即可查看效果。在端口占用的情况下，可以通过输入“hexo server -p 端口号”来改变端口号
