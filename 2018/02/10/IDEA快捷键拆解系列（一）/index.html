IDEA快捷键拆解系列（一）

  • 编辑器 快捷键
  • 2018-02-10

  这是IDEA快捷键拆解系列的第一篇文章。计划先从最顶部的每一个导航栏，以及中间区域的快捷键提示开始讲起。在此之前，请记住非常重要的一个快捷键： Alt + 下划线那个符号。举个栗子，在最左上角有++F++ile这么个导航项，所以对应的快捷键就是：Alt + F，其它任意有下划线的都是同样的操作，也包括导航项展开的任意子项，只要字符串中带有某一下划线的字符，你就可以通过这种形式来进行快速定位。当然了，除此之外，你还可以通过 ↑和 ↓来上下移动，然后通过Enter键来展开相应的子项。

  在整个IDEA编辑器中，中间区域的快捷键是最明显，所以应该也是最重要的，以下是对每一项的拆解介绍

中间区域快捷键提示区

作用 快捷键 拆解
Search Everywhere Double Shift 全局搜索，按两下Shift弹出此界面，再按两下Shift可以搜索非当前项目的文件，右上角还可以设置展示的类型。
Project View Alt + 1 Project面板的展开与折叠切换，还用于代码区快速跳转至项目区
Go to File Ctrl + Shift + N 搜素文件，按Ctrl+Shift+N弹出此界面，再按一下Ctrl+Shift+N可以搜索非当前项目的文件，右上角还可以设置搜索的文件类型。
Recent Files Ctrl + E 按Ctrl+E可以弹出记录了最近操作的面板，一般左边对应的是整个IDEA界面周边的各个功能选项，右边则是你最近操作的文件列表。左右方向键用于左右跳转，上下方向键用于切换，确定键用于打开。
Navigation Bar Alt + Home 跳转到项目导航条，也可以通过这种方式进行文件查找和打开。
